# Kairox
a decentralized talent exchange built on **Starknet**, designed to connect verified professionals with real opportunities at the right moment. By combining on-chain identity, reputation-based trust, smart contract workflows, and AI-assisted matching, Kairox rethinks how work is discovered, evaluated, and executed in a trust-minimized environment.

## Key Features

- **Opportunity Discovery**  
  Browse and explore roles published on-chain by companies and recruiters.

- **Role Exchange**  
  Create, manage, and publish opportunities through smart contractâ€“powered workflows.

- **Professional Identity**  
  Self-sovereign profiles backed by on-chain data and wallet-based authentication.

- **Reputation & Trust (Kairox Score)**  
  On-chain reputation reflecting verified work history and interactions.

- **AI-Assisted Matching**  
  Intelligent matching between opportunities and talent based on skills, history, and preferences.

- **Smart Contract Interactions**  
  Transparent, auditable workflows for role creation, applications, and agreements.

- **Starknet Wallet Integration**  
  Secure authentication and transaction signing using Starknet-compatible wallets.

## Technical Stack

- **Frontend:** TypeScript, Next.js,Tailwind CSS  
- **Backend:** NestJs  
- **Blockchain:** Starknet  
- **Wallets:** Both starknet-compatible and Eth wallets (e.g. Argent, Braavos, Mettamask)  
- **State & Data:** On-chain data + off-chain indexing
